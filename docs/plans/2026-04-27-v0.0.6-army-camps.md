# v0.0.6 Army, Camps, Hunger, and Game Over Implementation Plan

> **For agentic workers:** Use `/use-subagents` (preferred on capable harnesses) or `/execute` for batch checkpoints.

## Context

**Prompt:** “/start army size as a resource … v0.0.6: army size is a proxy for HP. Start with 5. The only way to lose it is to run out of food. Replenish army size in camps for free (random 1..2) with cooldown like farms. Make stepping on swamps and mountains cost 2 food instead of 1. Add game over when army reaches zero. Use sprite #100 for army. Camps have names + recruiting/empty lore pools. Signposts can point to camps too.”

**Reasoning:** Follow the v0.0.5 pattern: deterministic worldgen + pure reducer + tile enter handlers. Introduce camps as explicit PoIs in `world`, track cooldowns in `state.resources`, preserve determinism (no RNG on “empty/cooldown” lines), and keep rendering read-only. Keep code clean by sharing PoI helper utilities between farms and camps.

---

**Goal:** Implement v0.0.6 gameplay rules (army-as-HP, hunger damage, camps, terrain food costs, game over) with deterministic tests, and render army above food in the TIC-80 left panel.

**Architecture:** Core rules live in `src/core/*` (worldgen, reducer, tile enter handlers). `src/platform/tic80/*` renders the state and animations; it does not mutate state. Determinism is preserved by using `world.rngState` only in ready PoI outcomes (farm harvest, camp recruit).

**Tech Stack:** TypeScript, vitest, esbuild bundling to `the-unbound.js`.

**TDD during implementation:** enforce

TDD is enforced: for each behavior change, write a failing test first (RED), implement the minimal production change (GREEN), then refactor for clarity without changing behavior (REFACTOR).

In `src/core/signpost.ts`, update the file so it stays a single, pure formatter that selects the nearest PoI among **castle**, **farms**, and **camps**, with tie-break **Castle > Farm > Camp**, and with “same kind + same distance” keeping the earliest array order by **not** replacing `best` when ranks are equal.

```ts
import { dirLabel, manhattan, torusDelta } from './math'

export type PoiWorldView = {
  width: number
  height: number
  castlePosition: { x: number; y: number }
  farms: { position: { x: number; y: number }; name: string }[]
  camps: { position: { x: number; y: number }; name: string }[]
}

export function formatNearestPoiSignpostMessage(playerPos: { x: number; y: number }, world: PoiWorldView) {
  const candidates: { kind: 'castle' | 'farm' | 'camp'; name: string; pos: { x: number; y: number } }[] = [
    { kind: 'castle', name: 'The Castle', pos: world.castlePosition },
    ...world.farms.map((f) => ({ kind: 'farm' as const, name: `${f.name} Farm`, pos: f.position })),
    ...world.camps.map((c) => ({ kind: 'camp' as const, name: `${c.name} Camp`, pos: c.position })),
  ]

  const rank = (k: 'castle' | 'farm' | 'camp') => (k === 'castle' ? 0 : k === 'farm' ? 1 : 2)

  let best = candidates[0]!
  let bestDx = torusDelta(playerPos.x, best.pos.x, world.width)
  let bestDy = torusDelta(playerPos.y, best.pos.y, world.height)
  let bestD = manhattan(bestDx, bestDy)

  for (let i = 1; i < candidates.length; i++) {
    const c = candidates[i]!
    const dx = torusDelta(playerPos.x, c.pos.x, world.width)
    const dy = torusDelta(playerPos.y, c.pos.y, world.height)
    const d = manhattan(dx, dy)

    if (d < bestD) {
      best = c
      bestDx = dx
      bestDy = dy
      bestD = d
      continue
    }

    if (d === bestD) {
      // Tie-break: Castle > Farm > Camp. If same kind ties, keep earliest array order.
      if (rank(c.kind) < rank(best.kind)) {
        best = c
        bestDx = dx
        bestDy = dy
        bestD = d
      }
    }
  }

  const dir = dirLabel(bestDx, bestDy)
  return `${best.name}\n${dir}, ${bestD} leagues away.`
}
```

```ts
import type { Vec2 } from '../types'

export type NamedPoi = { position: Vec2; name: string }

export function findPoiIndexAt(pois: NamedPoi[], pos: Vec2): number {
  for (let i = 0; i < pois.length; i++) {
    const p = pois[i]!
    if (p.position.x === pos.x && p.position.y === pos.y) return i
  }
  return -1
}

export function pickDeterministicLine(lines: readonly string[], seed: number, poiIndex: number, stepCount: number): string {
  const m = lines.length
  if (m <= 0) return ''
  const k = ((seed | 0) + (poiIndex | 0) * 7 + (stepCount | 0)) | 0
  const idx = ((k % m) + m) % m
  return lines[idx] || lines[0] || ''
}
```

```ts
import { randInt } from '../prng'
import { CAMP_COOLDOWN_MOVES, CAMP_EMPTY_LINES, CAMP_RECRUIT_LINES, TILE_CAMP } from '../constants'
import { findPoiIndexAt, pickDeterministicLine } from './poiUtils'
import type { TileEnterHandler } from './types'

export const onEnterCamp: TileEnterHandler = ({ tileId, world, pos, stepCount, resources }) => {
  if (tileId !== TILE_CAMP) return { message: '' }

  const campIndex = findPoiIndexAt(world.camps, pos)
  if (campIndex < 0) return { message: '' }

  const campName = world.camps[campIndex]!.name || 'A Camp'
  const readyAt = ((resources.campNextReadyStep[campIndex] ?? 0) | 0) || 0

  if (stepCount < readyAt) {
    return {
      message: `${campName} Camp\n${pickDeterministicLine(CAMP_EMPTY_LINES, world.seed, campIndex, stepCount)}`,
    }
  }

  let rngState = (world.rngState | 0) >>> 0
  const rGain = randInt(rngState, 2)
  rngState = rGain.rngState
  const gain = (rGain.value | 0) + 1

  const rLine = randInt(rngState, CAMP_RECRUIT_LINES.length)
  rngState = rLine.rngState
  const line = CAMP_RECRUIT_LINES[rLine.value | 0] || CAMP_RECRUIT_LINES[0] || ''

  const nextCampNextReadyStep = resources.campNextReadyStep.slice()
  nextCampNextReadyStep[campIndex] = (stepCount | 0) + CAMP_COOLDOWN_MOVES

  return {
    world: { ...world, rngState },
    resources: {
      ...resources,
      armySize: (resources.armySize | 0) + gain,
      campNextReadyStep: nextCampNextReadyStep,
    },
    armyDeltas: [gain],
    message: `${campName} Camp\n${line}`,
  }
}
```

```ts
  const k = ((seed | 0) + (stepCount | 0)) | 0

  const idx = ((k % m) + m) % m
  return GAME_OVER_LINES[idx] || ''
}
```
