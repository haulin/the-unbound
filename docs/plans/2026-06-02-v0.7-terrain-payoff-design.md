## Overview

On entering **swamp** or **mountain**, the existing move-event roll runs first (lost / ambush / quiet). Only on **quiet**:

1. Roll **find** (~15%, keyed RNG on `{ seed, stepCount, cellId }`).
2. If find: roll **primary** and **secondary** amounts (independent, ¬±2 noise, floor 0).
3. Apply resources, **replace** the normal terrain flavor line with a **biome find** line describing the mixed haul.
4. Show **food** and **gold** delta flashes on the move.

### Move-step resource feedback (bugfix ‚Äî same milestone)

`reduceMove` already animates **food** and **army** deltas when `ENABLE_ANIMATIONS` is on. Any `TileEnterResult` that bumps `resources.food` (fishing lake, future find food) gets a food popup via `appliedFoodDelta`. **Gold gains on enter are not animated today** ‚Äî rainbow payout (+30 gold) updates inventory and lore but never calls `enqueueDeltas` for gold. Combat fights do enqueue gold inside the combat reducer, so the gap only shows on **overworld enter** rewards.

**Fix:** after `nextResources` is known, diff gold against `baseResources` and enqueue a gold delta on the move (same frame as food/army). That fixes rainbow retroactively and covers terrain finds without per-mechanic special cases.

**Acceptance add-on:**

```gherkin
GIVEN animations are enabled.
AND the player enters a rainbow end that has not paid out.
WHEN the step completes.
THEN a gold inventory delta animation plays for the payout.
```

Woods are unchanged. Finds do not open encounters. Food carry cap clamps food gains after payout (same as rainbow / combat).

### Reward shape (primary + secondary)

| Biome | Primary | Secondary | Diegetic read |
|-------|---------|-----------|----------------|
| Swamp | Food ~8 ¬±2 | Gold ~2 ¬±2 | Forage + picked-over coin |
| Mountain | Gold ~8 ¬±2 | Food ~2 ¬±2 | Cave hoard + sparse rations |

Independent rolls, each floored at 0 ‚Äî secondary may be 0 (same affordance as brigand food crumbs / goblin pocket change).

**Pattern (document in design; extract helper only if a third caller appears):** when both resources can drop, prefer **primary + secondary** over a single-resource table. Combat already does this (goblin food-primary, brigand gold-primary). Terrain find reuses the *shape* with fixed bases in `constants.ts`, not combat‚Äôs spawn-scaled formulas.

### Presentation

- **No** encounter modal, **no** preview plate (terrain has none today).
- **Lore:** one line from a **per-biome** pool (`SWAMP_FIND_LINES`, `MOUNTAIN_FIND_LINES`) written for a mixed haul (both resources may appear in copy even when secondary rolled 0 ‚Äî prefer lines that work when only primary matters, or split into ‚Äúbounty‚Äù vs ‚Äúscant‚Äù sub-pools if playtest copy feels wrong).
- **Deltas:** standard inventory food/gold popups on the move step. Reducer should diff `outcome.resources` vs `baseResources` for gold (and keep existing food path) so finds and rainbow payouts both flash correctly.

## Flow (quiet enter)

```
enter swamp | mountain
  ‚Üí rollMoveEvent (ambush / lost / quiet)
  ‚Üí if fight | lost: existing paths (unchanged)
  ‚Üí if quiet:
       ‚Üí if rollFind (15%): apply payout, SWAMP_FIND_* | MOUNTAIN_FIND_* message, return resources
       ‚Üí else: normal terrainLoreLinesForKind message
```

RNG: use `RNG.keyedIntExclusive` / stream helpers consistent with `rollMoveEvent` keys. **Do not** consume `world.rngState` for flavor; keyed draws only (RNG copy policy).

Find and ambush/lost are **mutually exclusive** on one step (find only after quiet). Food cost for the step still applies before `onEnterTile` (unchanged).

## Acceptance Specs (GWT)

```gherkin
;===============================================================
; Swamp quiet find pays food-primary + optional gold
;===============================================================
GIVEN the player stands on a swamp tile.
AND the move-event roll for that step is quiet (no lost).
AND the find roll succeeds.
WHEN the player completes the step onto that swamp.
THEN the lore message is a swamp find line, not generic swamp terrain flavor.
AND the player's food increased by at least 1.
AND the player's gold did not decrease.

;===============================================================
; Mountain quiet find pays gold-primary + optional food
;===============================================================
GIVEN the player stands on a mountain tile.
AND the move-event roll for that step is quiet (no ambush).
AND the find roll succeeds.
WHEN the player completes the step onto that mountain.
THEN the lore message is a mountain find line, not generic mountain terrain flavor.
AND the player's gold increased by at least 1.

;===============================================================
; Ambush suppresses find on the same step
;===============================================================
GIVEN the player enters a mountain tile.
AND the move-event roll for that step is an ambush.
WHEN combat opens.
THEN no terrain find payout was applied on that step.

;===============================================================
; Lost suppresses find on the same step
;===============================================================
GIVEN the player enters a swamp tile.
AND the move-event roll for that step is lost.
WHEN the player is teleported.
THEN no terrain find payout was applied on that step.

;===============================================================
; Woods unchanged
;===============================================================
GIVEN the player enters a woods tile on a quiet move-event outcome.
WHEN the step completes without an ambush starting.
THEN no terrain find payout from the swamp/mountain find system was applied.
```

**Spec count:** 5 specs covering 5 behaviors
