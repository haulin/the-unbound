# v0.8 Implementation Plan

> **Design:** `docs/plans/2026-06-03-v0.8-design.md`

**TDD during implementation:** enforce  
**ATDD during implementation:** enforce (8 GWT in design)

**Goal:** Healer + `buildOfferSets` + **placement RNG isolation** + rumor cap + lore-cycling + preview plates + Wyrm `'paid'` + TIC/lore polish.

**Non-goals:** Fisherman, Captain, Boar, Crossing, skip-empty-modal, hide hire buttons, **combat round keyed RNG** (follow-up), hover-focused preview plate.
